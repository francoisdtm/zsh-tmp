# zsh-tmp

This repository provides a `tmp` shell alias which creates a temporary directory and `cd` into it.

## Usage

    $ pwd
    /home/your_login
    $ tmp
    $ pwd
    /tmp/tmp.aFX8fenxPj

## Installation

Antigen:

    antigen bundle francoisdtm/zsh-tmp@main

Manual:

    source /path/to/zsh-tmp.plugin.zsh
